# Koi

All-in-one teen app for pure vibes and chillin.

This repository was converted to an Expo React Native project. To get started:

1. Install dependencies: `npm install` or `yarn`
2. Start Expo: `npm start` or `yarn start`
3. Run on device/emulator: choose Android or iOS from the Expo CLI

Notes:
- The app entrypoint is `App.tsx`.
- TypeScript configuration is included in `tsconfig.json`.
